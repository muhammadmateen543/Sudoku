## Sudoku Game (C++)

### Description
This is a console-based Sudoku game implemented in C++.  
The player fills a 9×9 Sudoku grid by entering values while following standard Sudoku rules.  
The program validates user input and checks whether the puzzle is solved correctly.

This project focuses on logical thinking, rule-based validation, and game-flow control.

### Features
- 9×9 Sudoku board
- User input validation
- Detects correct puzzle completion

### Tech Stack
- Language: C++
- Concepts: Arrays, Functions, Loops, Conditional Logic

### How to Run
1. Clone the repository
2. Open the project in a C++ compiler
3. Compile and run the program in console
